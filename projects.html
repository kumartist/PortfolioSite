Cortex Crusader Game Engine

Dlang SDL

The Cortex Crusader Game Engine supports both 2D and 2.5D (Mode 7) graphics. I worked with two teammates to build this engine from scratch using the D programming language and SDL as a part of the "Building Game Engines" course at Yale.

The engine allows you to create custom game objects and attach textures and behavior scripts. It also features a scene tree editor and saving/loading functionality for game scenes.

As a technical demo, my teammates and I built a humorous game titled "Mike's Mode 7 Massacre" in which you navigate a Mode 7 space while collecting capsules and avoiding a villain named Mike.

My contributions were the following:

  • Programming game object template and attachable components (scripts, textures, collisions)
  • Building the engine GUI
  • Implementing the scene tree editor
  • Implementing save/load functionality for game scenes
  • Creating art for the demo game
  • Programming movement and behavior of interactable objects within Mode 7 space (collectibles and enemies)

2D Solid Mechanics

C++ OpenGL

Implemented 2D collision physics for a custom triangle mesh, which involved the following steps:

  • Writing a stable, backward Euler timestepping technique for updating the mesh
  • Programming collision detection and the relevant response matrices
  • Integrating a stable Neo-Hookean energy to prevent the mesh from collapsing into itself. I derived the formulas by hand using calculus and linear algebra
  • Creating a custom triangle mesh
  • Implementing self-collision logic for each of the mesh vertices

I independently completed this assignment for the "Physics Simulation for Movies" Yale coruse taught by Professor Theodore Kim (former Pixar reseacher).

Smoke Simulation

C++ Open GL

Added vorticity confinement and Neumann boundary conditions to Jos Stam's Stable Fluids algorithm to simulate smoke.

Replaced the Gauss-Seidel solver with a matrix-free conjugate gradient solver to improve performance.

I independently completed this implementation as an assignment for the "Physics Simulation for Movies" Yale coruse taught by Professor Theodore Kim (former Pixar reseacher).

"Attack of the Cones" 3D Scene

Dlang Open GL

Created an interactive Star Wars inspired scene in OpenGL, which involved the following steps:

  • Creating a custom 3D model and textures for the "cone trooper" character in Blender
  • Writing a .obj file parser to load the 3D models
  • Leveraging instanced rendering to draw tens of thousands of characters and dust particles simultaneously
  • Writing a heightmap parser to render terrain data using triangle strips and apply color based on height
  • Implementing Phong shading for laser lights
  • Programming a camera system with tranlational and rotational controls
  • Adding a textured skybox
