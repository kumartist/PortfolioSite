Skater Gator (mobile game)

Unity C# XCode

Skater Gator was the first game I developed. For this project, I independently handled all aspects of development, including programming, art, animation, music, level design, release, and marketing.

The game was originally designed for web browsers and was hosted on my personal website. I then prepared the game for a mobile release by adding more stages, a cosmetics shop, saving functionality, a privacy policy, a store listing, and a video trailer. The game has since achieved over 3600 downloads from over 118 different countries.

Cortex Crusader Game Engine

Dlang SDL

The Cortex Crusader Game Engine supports both 2D and 2.5D (Mode 7) graphics. I worked with two teammates to build this engine from scratch using the D programming language and SDL as a part of the "Building Game Engines" course at Yale.

The engine allows you to create custom game objects and attach textures and behavior scripts. It also features a scene tree editor and saving/loading functionality for game scenes.

As a technical demo, my teammates and I built a humorous game titled "Mike's Mode 7 Massacre" in which you navigate a Mode 7 space while collecting capsules and avoiding a villain named Mike.

Terrainer

Unity C# MapBox SDK Avatar Maker

Terrainer is an experimental game which allows players to explore 3D terrain based on real-world satellite data. The game leverages the MapBox SDK for terrain and Avatar Maker for realistic human avatars.

I led a team of four to build a gameplay demo, and independently undertook all gameplay programming and character animation. The demo takes place in Yosemite valley and features a 3d character modeled after myself.

Monkey Fright

Unity C#

I had always been fascinated by PETA’s unique campaigns, which included web games. Believing that my own game development skills would be a valuable asset in such campaigns, I contacted the organization to begin building a web game raising awareness about monkey cruelty in health laboratories.

I handled all programming, art, and animation for the project while collaborating directly with PETA’s vice president of marketing to effectively target the campaign’s goals. The game was published on PETA’s website and social media.

Jogger Jam

Unity C# React Node.js

Jogger Jam is an in-development fitness game. It connects to your smartphone’s health app and rewards you with in-game currency corresponding to your calorie count. The awarded calories can then be redeemed to unlock fighter characters for a turn-based RPG.

For the project demo, I handled all programming, art, and animation for the mobile application. My teammates developed a companion web dashboard that would display players’ progress and provide them with additional lore related to the unlocked characters.
